# Insegnante teletrasportati - Teacher teleport yourself

Consente all'insegnante di teletrasportarsi in qualsiasi area STEM.
Allows the teacher to teleport themselves to any STEM area.

```sig
stemTeleporter.teacherTeleportSelf(STEMArea.Science)
```

Questo blocco è riservato agli insegnanti e consente di:
- Spostarsi rapidamente tra le diverse aree STEM
- Supervisionare le attività degli studenti
- Preparare le aree prima delle lezioni

This block is reserved for teachers and allows them to:
- Move quickly between different STEM areas
- Supervise student activities
- Prepare areas before lessons

## Parametri - Parameters

* **area**: L'area STEM di destinazione (Scienze, Tecnologia, Ingegneria, Matematica, Arte)
* **area**: The target STEM area (Science, Technology, Engineering, Mathematics, Arts)

## Utilizzo - Usage

L'insegnante seleziona l'area desiderata dal menu a tendina e viene immediatamente teletrasportato alla posizione corrispondente.

The teacher selects the desired area from the dropdown menu and is immediately teleported to the corresponding location.

```package
makecode-minecraft-stem-teleporter=github:username/makecode-minecraft-stem-teleporter
```
